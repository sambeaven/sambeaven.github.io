Adding Plugins to Yadr

Sep 6th, 2014 1:26 pm

So I installed yadr some time ago, but lo and behold I’d completely forgotten how to add plugins to vim. I knew it used vundle to handle plugins, but I wasn’t sure if there was a syntax they preferred.

Oh hey look, there is!

From this handy readme page on github:

yav -u https://github.com/airblade/vim-rooter

This automatically runs vundle’s :PluginInstall command to stick all the plugin gubbinz in their appropriate locations.
