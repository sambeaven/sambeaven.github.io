Vim-pencil for Soft Line Breaks

Sep 6th, 2014 1:31 pm

So I mentioned in a previous post that I wasn’t happy with the way that word wrapping was working in VIM for me. I’ve found a VIM plugin that manages the whole thing a lot easier, called VIM-pencil, which adds word-wrapping in a decidedly more sensible way. Mainly it does both hard and soft wraps without messing up the formatting of a post.

Thanks to YADR my environment is now looking very nice indeed.
